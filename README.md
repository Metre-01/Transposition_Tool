# Transposition-Tool-60fps-

A tool to help you transpose audio based on video frame count.

It helps you transpose audio in video editors, but **ONLY WORKS IN 60FPS MODE**.

It can calculate clip lengths between time and frames, e.g., 1s = 60 frames.

When in transposition mode, it calculates the target length, e.g., 120 frames + 12st = 60 frames.
